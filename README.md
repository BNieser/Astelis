# Astelis

Astelis is a modular, self-hosted media management system designed to bring order
to chaotic media libraries.

It automatically scans, identifies, and organizes your media files, enriches them with
metadata, and prepares them for further processing.

> 🚧 **Status:** Astelis is currently under active development. Features may change and the system is not yet production-ready.
>
> This repository acts as a mirror of the local development instance. Commits are pushed selectively when there is meaningful progress or features worth showcasing.
